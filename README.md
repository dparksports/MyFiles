# File Lister Utility
![Dashboard Screenshot](screenshot.png)

A high-performance file management and comparison tool designed for speed and reliability.

## Features
- **Fast Scanning**: Rapidly index drives and folders.
- **Checksum Calculation**: Compute SHA256 hashes for data integrity (buffered for low memory usage).
- **Comparison**: Compare file lists and checksum manifests to detect changes.
- **Real-Time Monitoring**: View file progress and RAM usage during operations.
- **Drive Selection**: Easily choose fixed or removable drives to scan.
- **Safe**: Read-only scan logic ensures your files are never modified.

## Usage
1.  **Select Drives**: Check the drives you want to scan on the left panel.
2.  **Scan**: Click "Scan Selected Drives".
3.  **Monitor**: Watch the progress bar and RAM usage.
4.  **Save**: The file list is automatically saved upon completion.
5.  **Checksums**: The application will automatically proceed to calculate checksums (can be paused/resumed).
